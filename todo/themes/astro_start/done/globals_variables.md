## Describe the feature

Public variables related to the product.

## Additional context

- Development-related variables are in the `package.json` file.
- Public variables related to the product are in the `src/globals.ts` file.
- Private variables related to the product are in the `.env` file.
- If possible, store private variables in the operating system and not in the application.
- Created based on previous projects.
